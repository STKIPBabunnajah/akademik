What are Training Tracks?

Each training track is designed to guide a student on which courses to enroll in, depending on his or her learning goals. O&B also includes recommendations on prerequisite and related courses.

Our Training Tracks

Java Developer Track

4 WEEKS
4 COURSES

The Java Developer Track is meant for computer science graduates who want to learn the multiple competencies needed to be productive in a real-world software development environment. This program is designed to help programmers gain a solid foundation in web programming, frameworks and software engineering, as well as the value, tools and techniques of software testing.

View Training Track

Agile Manager Track

3 WEEKS
4 COURSES

The Agile Manager Track is meant to provide a wide view of the methodologies and practices available in Agile, and to which situation each is applicable. It also tackles how Agile values affect requirements engineering by immersing participants in the entire project lifecycle - from initial project vision, to user roles, and to the first iteration and release.

View Training Track

Spring Developer Track

1 WEEK
2 COURSES

The Spring Developer Track is designed for experienced Java developers who would like to gain more depth on Spring, know how to build loosely-coupled and testable applications, as well as learn aspect oriented programming.

View Training Track

Automated Testing Track

4 WEEKS
11 COURSES

This track is meant for engineers who want to pursue a career in automation testing, and for quality assurance professionals who want to shift from manual testing to automation testing.

View Training Track

Enterprise Architect Track

3 WEEKS
4 COURSES

This track is designed to prepare senior software developers to assume the role of enterprise application architect, introducing him or her to the heuristics of designing and leading a software project for enterprise software projects. The heart of the course is the Enterprise Design Patterns course, a class that tackles known approaches to designing business applications.
